# crystal-libgit2

Git bindings for Crystal

There is a lot of work and testing todo on this one.

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  crystal-libgit2:
    github: puppetpies/crystal-libgit2
```


## Usage


```crystal
require "crystal-libgit2"
```
